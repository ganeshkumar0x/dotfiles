Dotfiles Symlink Script
======================

Quickly symlink dotfiles from this directory to your home directory.

Usage:
1. Make the script executable:
   chmod +x link.sh

2. Run the script:
   ./link.sh

Files that will be symlinked:
- .bashrc
- .img.jpg
- .tmux.conf
- .vimrc
- .xinitrc

Note: Existing symlinks will be overwritten.
